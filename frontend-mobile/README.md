# OrderFood Mobile Frontend

点餐系统移动端前端

## 技术栈

- Vue 3
- Vite
- Vue Router
- Pinia
- Vant 4
- Axios
- Swiper

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

### 3. 构建生产版本

```bash
npm run build
```

## 功能页面

- 首页 - 公告轮播、分类导航、推荐菜品
- 菜品列表 - 搜索、分类筛选、分页加载
- 菜品详情 - 详细信息、评价列表
- 购物车 - 菜品管理、数量调整
- 订单列表 - 订单查看、状态跟踪
- 预约点餐 - 时间选择、人数设置
- 个人中心 - 用户信息、订单管理
- 登录/注册 - 用户认证

src/
